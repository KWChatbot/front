# kwchatbot Front
챗봇 시연용 프론트페이지입니다.  

## 1.시작하기
시작하기 위해선 `npm` 혹은 `yarn`이 필요합니다. [yarn 링크](https://yarnpkg.com/lang/en/), [npm 링크](https://www.npmjs.com/)
```
git clone https://github.com/KWChatbot/front.git
cd front
yarn install
yarn start
```
